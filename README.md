# blog_django
Блог с возможностью регистрации и авторизации, изменении данных пользователя на Python с помощью фреймворка Django

Python - 3.9.2
Django - 3.1.7

Django site - Сайт-блог на Джанго.
Зарегестрироваться, авторизоваться, можно листать посты, изменять данные профиля.
Чтобы запустить в консоли прописать комманду: python manage.py - сайт работает локально на вашем пк, выйдет адрес страницы, которую нужно скопировать и вставить в поисковую строку вашего браузера. :)
